# 🔭 Dioptra FIDC

**Governança e Transparência para Fundos de Direitos Creditórios**

Dashboard automatizado de FIDCs de Duplicatas e PME com métricas de governança,
comparação com benchmarks (CDI/IMAB) e análise inspirada nos 3 pilares de Basileia.

> *"Dioptra" — do grego διόπτρα, instrumento para "ver através".*

## 📋 Abas

| Aba | Descrição |
|---|---|
| **0. Capa** | Disclaimer, instruções de uso |
| **1. Ranking Completo** | Todos os FIDCs com filtro automático |
| **2. Top 10 por Métrica** | Melhor fundo em cada item |
| **3. CDI/IMAB** | Comparação com benchmarks |
| **4. Governança** | Estatísticas + interpretação |
| **5. Propósito** | Ensaio reflexivo |
| **6. Fontes** | Documentação dos dados |
| **7. Glossário** | Definição de cada métrica |
